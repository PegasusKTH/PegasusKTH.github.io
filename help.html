Help/FAQ

What is PEGASEARCH?

PEGASEARCH is an easy tool for KTH students and staff to search for prerequisites of a certain course.

How to use PEGASEARCH?

Before using PEGASEARCH you need to install an addon called moesif CORS. It can be found in addons for Chrome and Firefox. Activate it before searhing for courses, and then deactivate it when done, since it creates security issues on other websites. Enter the course code of the course you want to search prerequisites for in the search box on the homepage and the prerequisites will be displayed both as a tree structure and as a list. We recommend using Chrome for our product instead of other browsers, since it has better performance and your search will be quicker.

Which courses can I search for on PEGASEARCH?

You can use PEGASEARCH to search for courses from The Royal Institute of Technology. PEGASEARCH supports courses in the formats: AB1234, ABC1234, AB123C, A1B2345, where A,B and C represent letters and 1-5 represents numbers. Example: IE1206, FAG3102, AE212X and F3E5051.

Can only KTH students and staff use this website?

No, our website is open for everyone, but it is most useful to KTH students and staffs to check prerequisites for specific courses. Others who would like to search for KTH courses are also welcome to use our tool, but only KTH courses can be found through this website.
