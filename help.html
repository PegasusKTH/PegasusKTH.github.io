Help/FAQ

xxx(Name of web) is an easy tool for KTH students and staffs to search for prerequisites of a certain course.

1. How to use xxx(Name of web)?

Enter the course code of the course you want to search prerequisites for in the search box on the homepage and
the prerequisites will be displayed both as a tree and as a list

2. Is it only KTH students and staffs can use this website?

No, our website is open for everyone but it is mostly useful for KTH students and staffs to check prerequisites of
a specific course. Other students who would like to know something about KTH courses are also welcome to use
this website but only KTH courses can be found through this website

3. I want to know more about this website

If you want to know more about this site, either browse through our project website, or contact us. (Contact info can
be found at the end of the page)
